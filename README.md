# PyconZA 2014 Website

Based on [wafer](https://github.com/CTPUG/wafer).

# Running

1. Checkout wafer to `../wafer`
1. (optional) Create a `localsettings.py` containing local settings.
1. `npm install`
1. `$(npm bin) bower install`
1. `bower install`
1. `./manage.py syncdb`
1. `./manage.py migrate`
1. `./manage.py runserver`
